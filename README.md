# Country Wiki

## About

Country Wiki is a quick country information lookup app

## Project Status

-- The project is still in development stage, will make more additions and improvements gradually.

## Description

### Use Case

-- A quick country general infromation look up,

## Instruction

--By entering a country name or choosing a country from a dropdown list of country names(unsorted), DOM will render country information like:Country name, flag image,coat of arms image,capital,continents, majority language,currency and a link
to google map of the country.

## Deployment
